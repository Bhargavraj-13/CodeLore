# Journey Page (Frontend)

The Journey Page allows users to explore explanations and learning journeys contributed by other users for a specific topic. It complements the main topic content by showcasing real-world understanding and perspectives. This page focuses on peer learning and shared knowledge.

## Features

- Tab-based navigation between Content and Journeys

- Displays user-written journeys linked to a topic

- Clean, readable layout optimized for learning

- Handles loading and empty states gracefully

## Tech Stack

- React (Functional Components & Hooks)

- React Router

- Axios for API communication

- Tailwind CSS for styling
